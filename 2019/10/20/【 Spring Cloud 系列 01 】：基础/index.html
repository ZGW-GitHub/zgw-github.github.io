一、SpringCloud

Spring Cloud 是一个 RPC ( Remote Procedure Call ) 框架 , 即 : 远程过程调用 , 它是一种通过网络从远程计算机程序上请求服务 , 而不需要了解底层网络实现的技术 .

Spring Cloud 为开发人员提供了一些工具，可以快速构建分布式系统中的一些常见模式 ( 例如 : 组态管理、服务发现、断路器、智能路由、微代理、控制总线、一次性令牌、全局锁、领导权选举、分布式会话、集群状态 ) 。
