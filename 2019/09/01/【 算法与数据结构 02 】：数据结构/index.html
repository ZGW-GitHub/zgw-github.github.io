⑤、红黑树

  • 每个节点为红色或黑色
  • 根节点为黑色
  • 每一个叶子节点（即：最后的 NULL 节点）为黑色
  • 如果一个节点是红色的，那么其孩子节点均为黑色
  • 从任意一个节点到叶子节点，经过的黑色节点是一样的（ 黑平衡 ）

2、B 树

  • 每个节点最多有 M 颗子树，M>2 ；
  • 除根节点外，其它每个分支节点至少有┌m/2┐棵子树 ；
  • 根节点至少有两颗子树（除非B树只包含一个节点）；
  • 所有叶子节点在同一层上。B树的叶节点可以看作是一种外部节点，不包含任何信息（这里的叶子节点含义：叶子节点只是没有孩子和指向孩子的指针）；
  • 有 j 个孩子的非叶子节点恰好有 j-1 个关键码，关键码按递增次序排列 。

①、2-3 树

  • 满足二分搜索树的基本性质
  • 节点可以存放 1个 或 2个 元素
  • 每个节点有 2个 或 3个 孩子
  • 2-3树是一颗绝对平衡的树
